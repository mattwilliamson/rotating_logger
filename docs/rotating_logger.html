Description

This is a plaintext logger which has the ability to rotate between log files when they are considered full. When it reaches the last file, it will open the first file.

handle_event/2

handle_event(Event, State) -> {ok, State} | {swap_handler, Args1, State1, Mod2, Args2} | remove_handler

Whenever an event manager receives an event sent using gen_event:notify/2 or gen_event:sync_notify/2, this function is called for each installed event handler to handle the event.

Preferrably called by gen_event:notify/2 e.g. gen_event:notify(event_logger, {Class, Msg}) Class could be any term, but the following are recommended: error | warning | info
